# PrimerTrabajophp

Este es el primer trabajo que hice con php, como se puede ver es simple y algunas cosas no van, están descritas en el fichero `leeme.txt`.

El funcionamiento de la aplicacion es el de un gestor de inventario de un supermercado ficticio, después de iniciar sesión el usuario puede añadir los productos a la tabla, ver un producto en concreto a través de una vista o ver el valor total del mercado.
